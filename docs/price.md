# 价格

由于TianliGPT有服务器维护成本和运算成本，所以此服务为付费服务。

到[HeoStore](https://store.zhheo.com?cid=1&mid=3)中购买，原价10元5万字符（Heo限量限时折扣9元）。请求过的内容再次请求不会消耗key，可以无限期使用。
